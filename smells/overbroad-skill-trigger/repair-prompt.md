# Repair Prompt: Overbroad Skill Trigger

Review the current `SKILL.md` for an overbroad trigger.

Make the smallest safe patch that narrows when the skill should activate. Do not add a new framework or rewrite unrelated skill content.

Please:

1. Identify trigger language that matches too many tasks.
2. Replace broad "use for any coding task" language with a specific use case.
3. Add clear "do not use" cases for nearby but different work.
4. Remove claims that this skill should run before all other skills.
5. Keep only instructions that support the narrowed use case.

After editing, summarize the old trigger, the new trigger, the non-goals added, and any remaining ambiguity.
